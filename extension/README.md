# CVBoost · Extensión de Chrome

Extensión oficial para leer la oferta de empleo de la pestaña actual y abrirla en CVBoost con un click.

## Instalación rápida (modo desarrollador, sin Chrome Web Store)

1. Abre Chrome y ve a `chrome://extensions`.
2. Activa el interruptor **"Modo desarrollador"** (arriba a la derecha).
3. Pulsa **"Cargar descomprimida"** y selecciona la carpeta donde has descomprimido este zip.
4. Listo: verás el icono de CVBoost en la barra del navegador.

Atajo de teclado: **Ctrl+Shift+B** (Mac: Cmd+Shift+B).

## Cómo se usa

1. Abre cualquier oferta de empleo en LinkedIn, Indeed, InfoJobs, Glassdoor o Welcome to the Jungle.
2. Pulsa el icono de CVBoost (o el atajo de teclado).
3. Pulsa **"Analizar esta oferta"** en el popup.
4. CVBoost se abrirá en una pestaña nueva con la oferta ya cargada. Solo te falta subir tu CV.

## Privacidad

- La extensión NO envía datos a servidores propios.
- Solo lee texto de la pestaña activa cuando tú pulsas el botón.
- El texto se pasa por URL a CVBoost y no se guarda en otro sitio.

## Publicación en Chrome Web Store

Cuando quieras subirla:

1. Crea una cuenta de desarrollador en https://chrome.google.com/webstore/devconsole/ (5 USD pago único).
2. Comprime esta carpeta en zip.
3. Sube el zip, rellena descripción, capturas y categoría.
4. Espera la revisión (1-3 días normalmente).
